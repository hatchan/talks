# Using gokit to build microservices

This talk will introduce the audience from the basic concepts, like pods and
replica controllers to more advanced concepts, like ingress. This talk is not
yet scheduled.

## Slides

The slides use [present](https://godoc.org/golang.org/x/tools/cmd/present) and
are located in the `/present` directory.

## Code examples

Any code examples will put in `/code` directory.

## Installing present

Make sure you have a `$GOPATH` setup, then do:

```
go get golang.org/x/tools/cmd/present
```
